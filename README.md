# 병원 사이트(메인)

1. Vanilla JavaScript를 활용한 인터랙션 기능 구현
    - 탭 전환, 버튼 이벤트, 슬라이더 동작 등 사용자 인터랙션 처리

2. Bootstrap 기반 헤더 네비게이션 구성
    - 반응형 레이아웃 적용
    - 모바일 환경 대응 네비게이션 구현

3. 자동 무한 슬라이드 애니메이션 구현
    - JavaScript 기반 자동 전환 로직 구현
    - PC / 모바일 환경에 따라 슬라이드 동작 방식 분기 처리

4. 카카오맵 API 연동을 통한 병원 위치 안내 기능 구현
    - 외부 지도 API 연동
    - 병원 위치 정보 시각화

5. 반응형 웹 디자인 적용
    - 미디어 쿼리를 활용한 PC / 모바일 화면 최적화
    - 모바일 환경에서 hover 이벤트 비활성화 처리

6. UX를 고려한 인터랙션 설계
    - 터치 디바이스에서 불필요한 hover 제거
    - 버튼 이중 클릭 발생 문제 개선
